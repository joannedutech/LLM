# LLM 個人知識庫

一個以個人網頁形式呈現的知識庫專案，用於整理與分享 LLM（大型語言模型）相關的學習筆記與資源。

## 專案簡介

本專案旨在建立一個結構化的個人網頁，彙整 LLM 領域的學習歷程、技術筆記與實用資源，方便日後查閱與知識分享。

## 功能規劃

- 學習筆記整理與分類
- LLM 相關技術文章收錄
- 實作範例與程式碼片段
- 常用資源與工具連結

## 快速開始

```bash
# 複製專案
git clone https://github.com/joannedutech/LLM.git

# 進入專案目錄
cd LLM
```
